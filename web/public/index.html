How to Calibrate the Mount

Calibration aligns the mount's internal model with the real sky. You need at least 3 reference objects (stars) with known coordinates. The more measurements you add, the better the accuracy.

1 Add reference measurements (Bootstrap)
  1. In the Initial Calibration (Bootstrap) card below, click "Add Reference Measurement" to expand the search section.
  2. Type a star name or catalog ID (e.g. Polaris, Sirius, Vega) and click "Search".
  3. Click on an object from the search results to select it — its coordinates auto-fill into the form.
  4. Click "Slew & Add Measurement" to slew the mount to that object and record the measurement, or "Add Measurement Direct" if the mount is already pointing at it.
  5. Repeat for at least 3 objects spread across the sky.
2 Run the initial calibration

Once you have enough measurements, click the "Run Initial Calibration" button in the Bootstrap card. The controller computes the alignment transform. The "TPOINT Ready" field will show Yes when the mount is ready for precise calibration.

You can track the Alignment Error and RMS Residual — lower values mean better alignment.

3 Add precise measurements (TPOINT)
  1. In the Precise Calibration (TPOINT) card, use the same search workflow to add more reference measurements.
  2. Add measurements for 10–20 objects distributed evenly across the sky for the best pointing model.
  3. Each measurement records the difference between the expected and actual position of the object.
4 Run TPOINT calibration

Click "Run TPOINT Calibration" to compute the full pointing model. The TPOINT model fits up to 6 error terms (cone, hub, collimation, etc.) and reports:

  • Residual RMS — typical pointing error after correction
  • Max Residual — worst-case error
  • Chi-Squared — overall model fit quality (lower is better)
5 Tips
  • Spread objects across the sky — avoid clustering them in one area.
  • Use the "Clear Measurements" button to reset and start over if needed.
  • Click "Refresh Status" to manually update the displayed statistics.
  • After calibration, use the Control tab's calibrated nudge buttons for fine adjustments.
  • Re-run calibration if you change the setup (new location, different load, etc.).
